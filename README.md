AI Engineering Resources & Notes
This repository contains a comprehensive collection of notes, guides, and resources focused on AI Engineering, specifically following the DataCamp AI Engineer Track.
📂 What's Inside?
The contents are organized into specific domains of Artificial Intelligence and Machine Learning:
Deep Learning & LLMs: Notes on neural networks, Large Language Models (LLMs), and frameworks like Hugging Face.
Generative AI: Folders dedicated to Llama LLM and Explainable AI (XAI).
Machine Learning Fundamentals: Comprehensive notes on Supervised and Unsupervised learning.
MLOps & Engineering: Resources on MLOps, Software Engineering for AI, Git version control, and Testing.
Responsible AI: Materials covering Responsible Data practices and Ethics in AI.
🛠️ Tech Stack Covered
Languages: Python
Frameworks: Hugging Face, PyTorch/TensorFlow (Deep Learning)
Tools: Git, MLOps Pipelines
